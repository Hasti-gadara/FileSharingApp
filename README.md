# File Sharing Application

A secure file sharing application built using Spring Boot, MySQL, and REST APIs.

## Features
- Upload files
- Download files
- Secure file access
- REST API support
- Database integration

## Tech Stack
- Java
- Spring Boot
- MySQL
- Maven
- REST APIs
